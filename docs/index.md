## üìù Edit Documentation

To create or edit documentation pages, use the **Phronidoc Editor**:

1. Start the editor service:
   ```bash
   cd editor-service
   ./start.sh
   ```

2. Open the editor in your browser:
   ```
   http://localhost:8080
   ```

3. Create new pages or edit existing ones directly in the web interface.

All changes are automatically committed and pushed to Git.
